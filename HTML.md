# HTML (HyperText Markup Language)

**HTML** — это язык разметки, который определяет **структуру и семантику** веб-страницы.

Он описывает:
- какие элементы есть на странице
- каков их смысл (заголовок, статья, кнопка и т.д.)
- иерархию контента

## Основные задачи HTML
- Семантическое описание контента
- Доступность (a11y)
- SEO
- Базовая структура UI

## Ключевые концепты
- HTML Elements
- HTML Semantics
- Document Structure
- Forms
- Accessibility

## HTML и другие технологии
- Визуальное оформление → [[CSS]]
- Логика и интерактивность → [[JavaScript]]
- Работа с DOM → DOM API
