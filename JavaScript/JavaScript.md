# JavaScript

**JavaScript** — это язык программирования, который добавляет **логику, интерактивность и поведение** веб-приложениям.

Он выполняется:
- в браузере
- на сервере (Node.js)
- в других runtime-средах
## Основные задачи JavaScript
- Управление DOM
- Работа с событиями
- Асинхронные операции
- Бизнес-логика приложения
