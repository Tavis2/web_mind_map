# (Cascading Style Sheets)

**CSS** — это язык стилей, который отвечает за **визуальное оформление** и **раскладку** HTML-документов.

Он определяет:
- внешний вид элементов
- позиционирование и layout
- адаптивность интерфейса
- анимации и переходы

## Основные задачи CSS
- Layout страницы
- Responsive design
- Поддержка разных устройств
- UI/UX оформление

## Ключевые концепты
- Box Model
- Selectors
- Specificity
- Cascade
- Flexbox
- Grid
- Media Queries

## Архитектура и подходы
- CSS Architecture
- BEM
- Utility-first CSS
- CSS-in-JS

## Связи
- Структура → [[HTML]]
- Динамические стили → [[JavaScript]]
- Производительность → Rendering Pipeline
